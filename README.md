OVERVIEW
========

gSimpleNotes is note app create in Gtk and Python

INSTALL
========

Download NotesCmd.py and run it in console

>python ./NotesCmd.py

USAGE
========

> R Read lines

> A Add lines

> D Delete lines

> Q Quit 

NOTE FILE SYNTAX
=================

> [1] Note

> [2] Note2

> @ comment

> [3] note3
